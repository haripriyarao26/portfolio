# Project Images

Place your project screenshots/images here.

## File Naming Convention:
- `moodbite.png` or `.jpg` - Screenshot of MoodBite application
- `project-name.png` or `.jpg` - Other project screenshots

## How to Add Images:

1. Save your project screenshots in this folder
2. Update the `image` property in `components/Projects.tsx` to point to the image:
   - Example: `image: "/projects/moodbite.png"`

## MoodBite Image:

Please add a screenshot of your MoodBite application as `moodbite.png` or `moodbite.jpg` in this folder.
